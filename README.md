# Tutorial Generator — 个性化教程生成器 Skill

一个 [Claude](https://claude.com) 技能(Skill):你说想学什么,它**先问几个简单问题**了解你的目标和水平,**联网查最新资料并核对**,然后产出一份**为你量身定制、深入详尽**的教程文档——带练习、自测、速查表、术语表、常见陷阱和可点击来源,还会联想底层机制与邻近学科帮你真正"想通"。

默认输出 Markdown,可选 Word(.docx)或 PDF。适用于 Claude 的 Cowork 模式与 Claude Code。

---

## 它解决什么问题

网上的通用教程不是为"你"写的——要么太浅、要么跳过你的盲点、要么用的是过时的工具和版本。这个技能把一位**好私教**的做法固化下来:上课前先问几个关键问题、课前做足功课查最新资料,再讲一份正好接住你的内容。

## 工作流程

1. **开场提问(默认必做)**
   - 先问基础:学习目标、当前水平、应用场景(秒出,无需等待)。
   - 再**联网查"大家学这个主题最常问、最常卡的点"**,据此做针对性摸底:点名具体技术问你"会不会",列出常见痛点问你"中不中招"。
   - 留一个开放问题挖你**真正的卡点**——它会成为整篇教程的主线。
   - 全程封顶约两轮、6–8 题,且随时可说"够了直接写"跳过。

2. **联网研究 + 核查**
   - 检索官方文档、当年的最佳实践、常见坑、版本/价格/模型名等易变信息。
   - **关键事实用第二来源交叉核对**,**代码示例尽量实跑验证**,没验证的明确标注。

3. **生成教程**(基于 `assets/tutorial_template.md` 骨架)
   - 结构:大局观 → 渐进式实战课 → 练习+答案 → 自测 Quiz → 速查表 → 术语表 → 常见陷阱 → 下一步/资源 → 来源。
   - 示例全部贴合你的应用场景;你点名的卡点作为主线深讲。

4. **按你选的格式交付**
   - Markdown(默认),或转成 Word / PDF;并可选设置**间隔复习提醒**。

## 特色

- **真正为你定制**:示例用你的领域,深度随你的水平校准。
- **永远最新**:联网查询并核对当前的工具、版本、价格、最佳实践。
- **专家高阶视角**:不止讲"怎么做",还联想到**底层机制(第一性原理)、邻近学科的成熟概念、跨域心智模型**,帮你想通而非死记。
- **完整教学闭环**:练习、自测、速查表、术语表、陷阱、进阶资源一应俱全。
- **多格式输出**:Markdown / Word / PDF。
- **尊重你的时间**:提问有上限、随时可跳过。

## 安装与使用

### 在 Cowork 模式
下载本仓库打包出的 `tutorial-generator.skill` 文件,在 Claude 桌面应用里点击 **Save skill / 安装** 即可。

### 在 Claude Code
把 `tutorial-generator/` 目录放进你的技能目录(例如 `~/.claude/skills/`),Claude Code 启动时会自动发现。

### 怎么触发
直接说出你的学习需求即可,例如:
- "教我怎么用 RAG,我是后端开发"
- "帮我做一份个人理财基金定投的入门教程"
- "我想学 AI 提示词工程,写份教程给我"

技能会先问你几个问题,然后开始。

## 仓库结构

```
tutorial-generator/
├── SKILL.md                      # 技能主体:流程与规则
├── README.md                     # 本文件
├── assets/
│   └── tutorial_template.md      # 教程结构骨架(每次从它起步)
└── evals/
    └── evals.json                # 测试用例(用于评测技能效果)
```

## 开发与评测

`evals/evals.json` 里是三个真实场景的测试用例(电商客服学提示词、后端开发学 RAG、上班族学理财),可配合 Anthropic 的 `skill-creator` 做带基线对比的评测与迭代。
